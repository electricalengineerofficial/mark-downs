## TIPS FOR EXECUTION

### Recording Tips
- Batch record similar videos on same day (save time & energy)
- Use templates for consistent setup
- Record Shorts while filming main content
- Always film B-roll/supplementary footage

### SEO Optimization Per Video
- Primary keyword in title (first 65 characters)
- 5-7 primary tags + 8-15 secondary tags
- Timestamps in description (critical!)
- 350+ word description with links
- Custom thumbnail designed beforehand

### Engagement Best Practices
- Pin comment with timestamps in first hour
- Respond to top 15-20 comments (engagement algorithm boost)
- Add cards/end screens linking to next logical video
- Monitor for common questions to answer in next video

### Analytics to Monitor
- Click-Through Rate (CTR) - target 8%+
- Average View Duration - target 50%+ of video length
- Impressions vs Clicks ratio
- Subscriber growth rate
- Watch time trends
